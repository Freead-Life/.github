# Freead Life 字由跳阅

Making Chinese reading accessible for everyone.
让每个人都能轻松阅读中文。

An intelligent typography platform for people with dyslexia, ADHD, and other neurodiverse reading needs. We optimize Chinese text layout to reduce cognitive load and improve reading comfort.

专为阅读障碍、ADHD 及神经多样性用户打造的智能排版平台。通过优化中文文本布局，降低认知负荷，提升阅读舒适度。

</div>

## Web Application — https://freead.life

- Paste or upload text, get optimized reading layout instantly
- Supports TXT, DOCX, PDF, EPUB, and images (with OCR)
- Export to PDF / Markdown

## WeChat Mini-Program 微信小程序

- Scan to open — no install needed
- Mobile-first reading experience

## FastAPI

RESTful API for third-party integration. Supports Web, APP, Mini-Program, and any authorized services. Contact admin for access of Swagger UI for testing.

---
## Who It's For

- Dyslexia 阅读障碍 — Clear visual boundaries, reduced line density
- ADHD — Rhythm-aware pacing that maintains attention
- General readers — A more comfortable Chinese reading experience
